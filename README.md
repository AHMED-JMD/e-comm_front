# EComm Platform - Frontend

منصة تجارة إلكترونية حديثة وسريعة وآمنة مبنية بـ React و Vite

## 🚀 المميزات

- ✨ تصميم حديث وجذاب مع Tailwind CSS
- ⚡ أداء سريع جداً مع Vite
- 📱 واجهة متجاوبة وصديقة للهاتف
- 🌙 دعم اللغة العربية
- 🔐 نظام مصادقة آمن
- 🎨 مكونات قابلة لإعادة الاستخدام
- 📦 بنية منظمة وقابلة للتوسع

## 📋 المتطلبات

- Node.js 16+
- npm أو yarn

## 🛠️ التثبيت

```bash
# انتقل إلى مجلد المشروع
cd e-comm_front

# ثبت الحزم المطلوبة
npm install

# أو استخدم yarn
yarn install
```

## 🚀 تشغيل المشروع

### في بيئة التطوير:

```bash
npm run dev
```

سيتم فتح التطبيق على `http://localhost:3000`

### بناء المشروع للإنتاج:

```bash
npm run build
```

### معاينة بناء الإنتاج:

```bash
npm run preview
```

## 🎯 الصفحات المتاحة

### 1. الصفحة الرئيسية (`/`)

- عرض الفئات الرئيسية
- عرض المنتجات المميزة
- شريط بحث سريع
- إحصائيات المنصة

### 2. صفحة تسجيل الدخول (`/login`)

- نموذج تسجيل دخول آمن
- خيار تذكري
- رابط نسيان كلمة المرور
- خيارات تسجيل اجتماعي

### 3. صفحة التسجيل (`/register`)

- نموذج تسجيل شامل
- اختيار نوع الحساب (مشتري/بائع)
- التحقق من صحة البيانات
- قبول الشروط والأحكام

## 🎨 المكونات الرئيسية

### Header (الرأس)

- شعار المنصة
- قائمة التنقل
- أيقونة سلة التسوق
- أزرار الدخول والتسجيل
- قائمة جوال متجاوبة

### Footer (التذييل)

- روابط سريعة
- السياسات والشروط
- وسائل التواصل الاجتماعي
- بيانات الاتصال

## 🔐 الميزات الأمنية

- التحقق من صحة نماذج الإدخال
- تشفير كلمات المرور
- حماية بيانات المستخدم
- معالجة آمنة للأخطاء

## 🎨 التخصيص والألوان

يمكن تخصيص الألوان من خلال ملف `tailwind.config.js`:

```js
theme: {
  extend: {
    colors: {
      primary: '#3B82F6',      // أزرق
      secondary: '#10B981',    // أخضر
      accent: '#F59E0B',       // أصفر
      dark: '#1F2937',         // رمادي غامق
      light: '#F3F4F6',        // رمادي فاتح
    },
  },
}
```

## 🔧 إضافة صفحات جديدة

1. أنشئ ملف JSX جديد في `src/pages/`
2. أضف المسار في `App.jsx`:

```jsx
<Route path="/your-page" element={<YourPage />} />
```

3. أضف الرابط في الـ Header إذا لزم الأمر
